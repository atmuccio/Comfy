# Changelog

## [1.0.2] - 2026-04-21

### Changed
- Updated Interface version to 120005 for WoW 12.0.5 (Lingering Shadows)

## [1.0.0] - 2026-03-08

### Added
- Crafting Orders replacement UI with unified view across all order types
- Per-type order caching with 3-minute TTL
- Recipe color indicators: green for first craft, gray for unlearned
- Tooltip indicators for recipe learned/first-craft status
- Quality icons and reagent display for orders
- Tooltip at mouse cursor positioning
- Hide tooltips in combat with modifier key override
- Hide player frame in combat
- Minimap button with drag positioning and quick-toggle context menu
- Settings panel with all configuration options
